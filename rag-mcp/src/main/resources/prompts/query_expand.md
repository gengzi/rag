### 分场景扩写逻辑（辅助精准扩写）
#### 1. 生活咨询类（如“怎么缓解失眠”“周末短途旅行推荐”）
- 扩写方向：细化场景条件 + 关联过往经历 + 明确个性化需求
- 示例：
  原始问题：“怎么缓解失眠”
  扩写1：“你失眠的情况是偶尔发生，还是长期持续的？”
  扩写2：“平时是否有喝咖啡、熬夜等可能影响睡眠的习惯？”
  扩写3：“更倾向通过饮食、运动还是放松方式来缓解失眠？”

#### 2. 产品推荐类（如“推荐平价护肤品”“选什么型号的平板电脑”）
- 扩写方向：明确核心诉求 + 细化使用场景 + 界定偏好边界
- 示例：
  原始问题：“推荐平价护肤品”
  扩写1：“你是想改善肤质问题，还是单纯需要基础保湿类护肤品？”
  扩写2：“日常护肤更看重成分安全，还是使用后的肤感体验？”
  扩写3：“是否有需要避开的成分或偏好的护肤品牌类型？”

#### 3. 服务需求类（如“预约健身课程”“咨询留学申请”）
- 扩写方向：确认服务核心 + 细化时间/预算条件 + 关联后续需求
- 示例：
  原始问题：“咨询留学申请”
  扩写1：“你计划申请哪个阶段的留学，以及意向国家或地区是？”
  扩写2：“留学申请的预算范围和时间规划大概是怎样的？”
  扩写3：“是否有明确的专业方向，或对学校排名有具体要求？”

#### 通用约束
- 扩写问题不涉及复杂专业术语，确保用户易理解、能回应。
- 3个问题分别对应“是什么（核心）- 怎么样（场景/习惯）- 要什么（偏好/需求）”，全面覆盖潜在记忆关联点。
- 不偏离原始问题主题，不添加无关提问，所有扩写均为挖掘记忆关联服务。
